# Objective
The objective of this project is to design and develop an Executive Sales Performance Dashboard using Power BI that provides senior leadership with a clear, consolidated view of business performance.

Specifically, this project aims to:

- Transform raw transactional sales data into a star schema optimized for analytical reporting
- Implement core sales and profitability KPIs using DAX
- Enable time-based performance analysis, including Year-over-Year (YoY) and Month-over-Month (MoM) comparisons
- Build an interactive and executive-friendly dashboard that supports high-level monitoring as well as detailed drill-down analysis
- Demonstrate strong fundamentals in data modeling, DAX, and business storytelling

# Expected Outcomes
Upon completion, this dashboard will enable stakeholders to:

- Quickly assess overall sales and profit performance
- Monitor growth trends over time using YoY and MoM metrics
- Identify underperforming regions, markets, and product categories
- Drill through summary KPIs to analyze regional and product-level details
- Make data-driven decisions based on a single, consistent analytical view of sales data
